Posibles problemas:
1. El z-index podría estar interfiriendo
2. Elementos padre con overflow: hidden podrían cortar la sombra
3. El background del contenedor podría estar ocultando la sombra
4. Clases de Tailwind podrían estar sobrescribiendo el CSS
